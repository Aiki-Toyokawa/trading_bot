## 売買シグナル
### 買いシグナル
まとめ: 上昇トレンド + RSIが適正かつ上向き + ATRが適度 + Claudeが危険視していない + 最終スコアが十分高い銘柄
  1. 価格帯
     $8〜$250 の範囲外は除外です。
  2. 平均出来高
     20日平均出来高が閾値未満だと除外です。今の基準値は MIN_AVG_VOLUME = 80,000 ですが、iex フィードでは補正が入ります。
  3. Claudeの危険判定
     avoid=true は即除外、event_risk=high も除外です。
  4. トレンド条件
     基本は SMA50 > SMA200 の上昇トレンドのみです。
     ただし paper緩和モードでは、market_regime=trend かつ breakout かつ confidence が十分なら救済通過します。
  5. RSI条件
     RSI_LOW < RSI_now < RSI_high かつ RSI_now > RSI_prev です。
     通常は 40 < RSI < 65、強トレンド時は 40 < RSI < 75 まで許容します。
     paper緩和モードでは 35〜80 の救済もあります。
  6. レジームとconfidence
     market_regime=trend なら confidence >= 0.45、
     レンジ判定ならより厳しく confidence >= 0.72 が必要です。
  7. ATR条件
     ATR比率が低すぎても高すぎても除外です。
     今は 0.8% <= ATR/価格 <= 12% の範囲に入っている必要があります。
  8. スコア条件
     最終的に Python がスコアを組みます。主な加点は:
  - ブレイクアウト
  - 出来高急増
  - Claude confidence
  - sentiment
  - trend regime
  - SMAトレンド強度
  - ATRの好ましさ
  この合計が ENTRY_MIN_SCORE = 0.55 未満なら除外です。

  9. 実発注の追加閾値
     通過候補の中でも、実際に注文を出すには ENTRY_ORDER_MIN_SCORE 以上が必要です。
     今は通常 0.60、paper緩和時は 0.56 です。

### 売りシグナル
まとめ: -3%損切り / 前日比急落は即撤退 / +6%で半分利確 / 残りは高値追随トレーリング
  1. 初期損切り
     現在価格 <= entry_price * STOP_LOSS_RATE で全量売りです。今の設定だと STOP_LOSS_RATE = 0.97 なので、エントリー比 -3% で損切りします。
  2. 急落退出
     前日終値比で -6% 以下なら crash_exit で全量売りです。これは通常の損切りより優先される急落保護です。
  3. 半利確
     まだ半利確していない状態で、現在価格 >= entry_price * TAKE_PROFIT_RATE になったら半分売ります。今は TAKE_PROFIT_RATE = 1.06 なので、+6%到達で半分利確です。
  4. 半利確後のトレーリング売り
     半利確後は trailing_high_price を更新しながら、最高値から一定率下落したら残りを売ります。
     通常は TRAILING_RATE = 0.97 なので 高値から -3% です。
  5. 急落気味ならトレーリングをタイト化
     前日終値比 -2% 以下なら、トレーリング幅を 0.985 に締めます。つまり 高値から -1.5% で落ちたら売る防御モードです。
  6. 強トレンドなら利を伸ばす
     半利確後、SMA20 > SMA50 かつ価格が SMA20 以上で強トレンドなら、トレーリングを少し緩めます。
     +10% 以上で 0.965、+20% 以上で 0.955 まで緩和します。つまり 高値から -3.5% / -4.5% まで許容して利益を伸ばします。
  7. 残り半分の最低利益フロア
     半利確後の残りは、単純なトレーリングだけでなく、最低でも +1% 利益を守る床があります。含み益が大きいときは
     この床を +3% まで引き上げます。

  実運用上はさらに、既に未約定のSELL注文がある銘柄は新しい売り注文を出さない ので、重複売りも防いでいます。

## 用語メモ
RSI: (Relatice Strength Index) 相対力指数
70以上 → 買われすぎ（下落警戒）
30以下 → 売られすぎ（上昇警戒）

スリッページ: 表示価格と実際の約定にズレが生じること
投資家有利なポジティブスリッページと投資家不利なネガティブスリッページがある

ケリー基準: Kelly Criterion
長期的・複利的に資産を最大化するための最適な投資(賭け)比率を導き出す数式

SMA: 単純移動平均線 Simple Moving Average
株価などの一定期間の平均価格を線でつないだテクニカル分析指標
※ラップ口座 SMA(Separately Managed Account): 証券会社が投資家の一任を受けて運用・管理する個別勘定口座サービス

ナンピン(難平): 値下がりした際にさらに買い増しを行うこと。平均購入単価を引き下げる投資手法、下落継続で損失拡大

アルゴ: アルゴリズム取引(自動売買)の略称

ATR(Average True Range): ボラティリティを測る指標。ATR高いと値動き激しい、ATR低いと値動き小さい

Trend Regime: 市場の状態を分類する概念。トレンド相場(Uptrend, Downtrend)やレンジ相場(Sideways)など

リスクリワード: 損益比率、1回のトレードで取れるリスクに対する見込みリワード = 勝ちトレード平均利益/負けトレード平均損失

Breakout: 重要な価格帯を突破すること
